Quick Start for Developers
--------------------------

Setup\*\*
~~~~~~~~~

.. code:: bash

    git clone https://github.com/HaoZeke/pychum
    cd pychum
    uv sync --all-extras

Run Tests\*\*
~~~~~~~~~~~~~

.. code:: bash

    uv run pytest

Add New Engine\*\*
~~~~~~~~~~~~~~~~~~

1. Create ``pychum/engine/<code>/`` directory

2. Add \_dataclasses.py, \_renderer.py

3. Add to pychum/engine/\_\ :sub:`init`\\_\ :sub:`.py`\

4. Add tests in ``tests/engine/<code>/``

See `Input Generation <input_generation.rst>`_ for details.
