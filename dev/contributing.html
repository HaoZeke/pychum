Contributing to pychum¶

Getting Started¶

1. Fork and Clone¶

git clone https://github.com/your-username/pychum
cd pychum

2. Set Up Environment¶

uv sync --all-extras

3. Run Tests¶

uv run pytest

Code Style¶

Formatting¶

uv run ruff format .
uv run ruff check .

Directory Structure¶

pychum/engine/<code>/

|-- _renderer.py

  2. Add tests
