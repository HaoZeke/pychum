* Contributing to pychum

** Getting Started

*** 1. Fork and Clone

#+begin_src bash
git clone https://github.com/your-username/pychum
cd pychum
#+end_src

*** 2. Set Up Environment

#+begin_src bash
uv sync --all-extras
#+end_src

*** 3. Run Tests

#+begin_src bash
uv run pytest
#+end_src

** Code Style

*** Formatting

#+begin_src bash
uv run ruff format .
