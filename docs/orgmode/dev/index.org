* Quick Start for Developers

** Setup**

#+begin_src bash
git clone https://github.com/HaoZeke/pychum
cd pychum
uv sync --all-extras
#+end_src

** Run Tests**

#+begin_src bash
uv run pytest
#+end_src

** Add New Engine**

1. Create =pychum/engine/<code>/= directory
2. Add _dataclasses.py, _renderer.py
3. Add to pychum/engine/__init__.py
4. Add tests in =tests/engine/<code>/=

See [[file:input_generation.org][Input Generation]] for details.
