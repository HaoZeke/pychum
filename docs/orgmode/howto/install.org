* Installation Guide

** Quick Install

#+begin_src bash
pip install pychum
#+end_src

** Requirements

- Python 3.10, 3.11, or 3.12
- pint (unit handling)
- jinja2 (template rendering)
- tomli (TOML parsing)

** Platform Notes

- Linux: Full support
- macOS: Full support
- Windows: Full support

** Verification

#+begin_src bash
python -c "import pychum; print(pychum.__version__)"
#+end_src
